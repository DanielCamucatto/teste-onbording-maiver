# Central de Onboarding Maiver

> **Versão:** 1.0.0
> **Status:** ✅ MVP Completo

## 📋 Sobre o Projeto

Sistema interno para gerenciamento de onboarding de novos clientes da Maiver.

|-----------|------|

## 🛠️ Stack

- **Frontend:** React + Vite + TypeScript
- **Backend:** NestJS + TypeScript
- **Banco:** Supabase (PostgreSQL)

## 📦 Instalação

```bash
# Clone o repositório
git clone ...

# Backend
cd backend
npm install
npm run start:dev

# Frontend
cd frontend
npm install
npm run dev
```

## 🧪 Testes

```bash
# Backend
cd backend && npm test

# Frontend
cd frontend && npm test
```
